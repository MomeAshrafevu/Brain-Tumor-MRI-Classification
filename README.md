# Brain Tumor MRI Classification using Hybrid Deep Learning Pipeline

This repository contains the source code for a standardized and interpretable hybrid deep learning pipeline designed for Brain Tumor MRI classification.

## 🚀 Key Features
* **Hybrid Model Architecture:** Combines powerful backbones for feature extraction and optimal classification performance.
* **Explainable AI (XAI):** Integrated with Grad-CAM to visualize and interpret the model's decision-making process on MRI slices.
* **Comprehensive Metrics:** Includes implementation for generating Confusion Matrix, Per-Class Metrics Bar Chart, and ROC Curves (Micro & Macro AUC).

## 📊 Dataset
The model is trained and tested on the Brain Tumor MRI Images dataset.

## 📈 Results
The pipeline achieves high classification metrics, including:
* Micro AUC: ~0.9996
* Macro AUC: ~0.9997
